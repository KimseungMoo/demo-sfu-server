🔍 분석 결과

  • 선형적 증가 패턴: 스트림 개수가 늘어날수록 CPU 사용량이 선형적으로 증가

  • 성능 저하: 스트림이 늘어날수록 FPS와 처리 속도가 크게 저하
  • 최적 스트림 수: 현재 시스템에서는 1-2개 스트림이 최적

💡 권장사항
